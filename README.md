# TP_Python_Paises

Descripción del Programa
Este proyecto consiste en una aplicación desarrollada en Python para la gestión de información de países. El sistema permite cargar información desde un archivo CSV, realizar búsquedas, aplicar filtros, ordenar registros y generar estadísticas sobre los datos almacenados.

Los datos de cada país incluyen:

Nombre
Población
Superficie
Continente

La información se almacena en una lista de diccionarios y se mantiene actualizada mediante la lectura y escritura de un archivo CSV.

Funcionalidades
Cargar datos desde un archivo CSV.
Agregar nuevos países.
Actualizar población y superficie de países existentes.
Buscar países por nombre.
Filtrar países por continente.
Filtrar países por rango de población.
Filtrar países por rango de superficie.
Ordenar países por nombre.
Ordenar países por población.
Ordenar países por superficie.
Mostrar estadísticas generales.
Guardar automáticamente los cambios realizados.

Tecnologías Utilizadas
Python 3
Archivos CSV
Visual Studio Code
GitHub

Instrucciones de Uso
-Requisitos
Tener instalado Python 3.
Contar con el archivo paises.csv en la misma carpeta que el programa.
Ejecución:
Abrir una terminal en la carpeta del proyecto y ejecutar y luego seleccionar la opción deseada desde el menú principal.
